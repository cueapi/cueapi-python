# Security

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Email support@vector.build with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and aim to release a fix within 14 days for confirmed vulnerabilities.

## Supported versions

| Version | Supported |
|---|---|
| 0.1.x | Yes |
